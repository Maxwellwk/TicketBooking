某麦网抢票软件
=======
### 说明
这是某麦网的一个抢票软件，实现了抢票的基本功能，代码很简单，其他功能大家可以自行优化。
### 技术
实现本抢票软件所使用的技术是 pyppeteer，安装方式很简单 pip install pyppeteer 即可。使用的框架是scrapy，让pyppeteer嵌入到scrapy框架中执行。
